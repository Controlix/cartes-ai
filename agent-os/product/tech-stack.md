# Product Tech Stack

## Core Framework
- **Framework:** Next.js
- **Language:** TypeScript
- **Platform:** Web / Progressive Web App (PWA)

## Frontend
- **Library:** React
- **Styling:** Tailwind CSS (Recommended for rapid UI development)
- **Icons:** Lucide React
- **State Management:** React Context API + Hooks

## Data & Storage
- **Persistence:** LocalStorage (for saving game state on the device)
- **Offline Support:** Service Workers (via `next-pwa` or similar)

## Development & Quality
- **Linting:** ESLint
- **Formatting:** Prettier
- **Testing:** Jest, React Testing Library
