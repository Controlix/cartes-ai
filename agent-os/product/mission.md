# Product Mission

## Pitch
Belot Score Keeper is a Progressive Web App (PWA) that helps casual Belot players eliminate manual scorekeeping errors and hassle by providing an intuitive, mobile-first interface for tracking game progress and calculating results.

## Users

### Primary Customers
- **Casual Players:** Individuals playing friendly games of Belot who want a quick and accurate way to track scores on their mobile devices.

### User Personas
**The Social Card Player** (25-55)
- **Role:** Game Participant
- **Context:** Social gatherings, cafes, or family game nights.
- **Pain Points:** Slow mental math, disputes over scores, lack of pen and paper.
- **Goals:** A smooth gaming experience where scorekeeping is invisible and accurate.

## The Problem

### Manual Scorekeeping Friction
Traditional tracking with pen and paper is error-prone, slow, and requires physical supplies. It distracts players from the flow of the game.

**Our Solution:** A dedicated PWA that handles all calculations, enforces Belot scoring rules, and persists game state.

## Differentiators

### Dedicated PWA Experience
Unlike generic calculator apps, we provide a tailored Belot interface that runs natively on iOS and Android via the browser. This results in a seamless, app-like experience without the need for app store downloads.

## Key Features

### Core Features
- **Round Score Entry:** Intuitive input fields to record points for each team per round.
- **Real-time Totals:** Automatic calculation and display of the running total score for the game.

### Advanced Features
- **Game Submission:** Ability to finalize a game, declare a winner, and submit the final results.
